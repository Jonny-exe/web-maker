Add items: click the plus icon on the left and choose which item you want to add. You options are: title, subtitel, header, text, image, table and collumns.
Remove items: long press the item you want to remove and click remove item on the bottom.
Move items: long press the item you want to move and then select its new location by pushing the bar that is in its position.
Preview: to preview you created web page click the eye icon in the top middle. To unpreview click the eye again
Change project style: to change project style go to settings -> style, and choose your prefered styles
Change project title: to change project title go to settings and change the project title
Save your project: go to user settings on the top left. Then go to create project if you havent and follow the steps. Once you did it just click save.
Import a project: go to user settings -> import and write your project token
Export a project: go to user settings -> get html file, and follow the instructions.
